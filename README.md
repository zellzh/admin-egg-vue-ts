# Admin-Vue-Egg-TS
基于 Vue，Egg 开发的前后端分离的通用 Admin 系统

## 目录结构
+ backend-egg：egg-ts 开发的 Admin 后端项目
+ frontend-egg：vue-ts 开发的 Admin 前端项目
